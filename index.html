Musical RPG

Created by Michael Auld and Joshua Singer

Controls

Move Plink using the Arrow Keys

Press Note Buttons by walking up to them and pressing Space

Notes

You may like to zoom in, the game window is small.

The second room is HARD! We reccomend using a pen and paper to write out the pattern.

The first half second room's note pattern is the same as the note pattern from the first room.

A tune will play if you get the first half right, letting you know you are on track.

Requirements

Game Requires WebAudioAPI Support (some Safari versions will not work)

Game plays best in Google Chrome Web Browser
